# POSQuest — IGST Place of Supply Game

An interactive web app to master **Sections 9–13 of the IGST Act, 2017** through gamified scenarios.

## Features
- 🔍 Search-driven scenario selection covering all sub-sections of Sec 9–13
- 🗺️ Interactive SVG world map with accurate India (all states, UTs, J&K, Arunachal Pradesh, Ladakh)
- 🌊 Click territorial waters → Section 9 auto-triggers
- 🌍 Country shading on map for supplier & recipient locations
- 🎯 Quiz-style POS selection with instant legal feedback
- ☀️🌙 Light / Dark / Weather-based themes
- 📱 Fully responsive

## Sections Covered
| Section | Topic |
|---------|-------|
| 9 | Territorial Waters |
| 10(1)(a–e) | Goods — movement, bill-to-ship-to, no movement, installation, on-board |
| 11(a–b) | Import and Export of Goods |
| 12(2–13) | Services (both parties in India) — general, immovable property, restaurant, training, events, transport, telecom, banking, insurance |
| 13(2–8) | Cross-border services — general, physical performance, immovable, events, banking, intermediary |

## Deploy to GitHub Pages

1. Fork / clone this repo
2. Go to **Settings → Pages**
3. Set **Source** to `main` branch, `/ (root)` folder
4. Visit `https://yourusername.github.io/igst-pos-game/`

No build step. Pure HTML/CSS/JS — zero dependencies.
